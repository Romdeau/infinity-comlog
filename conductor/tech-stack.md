# Tech Stack

## Core Technologies
- **Framework:** React (v19)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Package Manager:** Bun

## UI & Styling
- **CSS Framework:** Tailwind CSS (v4)
- **Component Library:** shadcn/ui (based on Radix UI)
- **Icons:** Lucide React
- **Charts:** Recharts

## Routing & State
- **Routing:** React Router DOM (v6)
- **State Management:** React Context API

## Deployment & Tooling
- **Linting:** ESLint
- **Formatting:** TypeScript-ESLint
